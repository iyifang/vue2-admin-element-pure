# vue2-admin-element-pure

> vue2+element的一个节简版后台项目

**Git address：** https://github.com/z245580321/vue2-admin-element-pure

## 项目初始化
```bash
# 克隆地址
git clone https://github.com/z245580321/vue2-admin-element-pure.git

# 进入项目文件地址
cd vue2-admin-element-pure

# 安装依赖
npm install 或 npm i

# 运行项目
npm run dev
```
